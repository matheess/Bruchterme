Gleichungen – Aufgaben (Komplexität 2)
Bestimme die Lösung der Gleichung. Mache die Probe.

Aufgabe 1

a)\(\displaystyle \frac{4}{5}\left(t - \frac{5}{2}\right) - \frac{2}{5}\left(t - \frac{5}{3}\right) = 0\)★★☆☆☆
b)\(\displaystyle \frac{1}{2}\left(\frac{5}{2}n - \frac{3}{4}\right) - \frac{3}{4}\left(\frac{4}{3}n + \frac{4}{3}\right) = 0\)★★☆☆☆
c)\(\displaystyle \frac{5}{3}\left(\frac{3}{2}z - 6\right)^2 + \frac{3}{2}z + 6 = 15\left(\frac{1}{2}z - 2\right)^2 + \frac{1}{2}z + 4\)★★★☆☆

Aufgabe 2

a)\(\displaystyle \frac{3}{5}\left(\frac{3}{4}t - 3\right)^2 + \frac{4}{3}t - 5 = \frac{3}{5}\left(\frac{2}{3} + \frac{3}{4}t\right)^2 + \frac{3}{2}t + 1\)★★★☆☆
b)\(\displaystyle \frac{2}{3}z - 4 = \frac{5}{3}\left(-\frac{5}{2} + \frac{3}{5}z\right)^2 - \frac{5}{3}\left(-\frac{3}{2} + \frac{3}{5}z\right)^2\)★★★☆☆
c)\(\displaystyle \frac{3}{2}\left(5 + \frac{3}{5}y\right)^2 + \frac{4}{3}y + 3 - \frac{3}{5}y = \frac{27}{8}\left(\frac{2}{5}y + 2\right)^2 + 1\)★★★☆☆

Aufgabe 3

a)\(\displaystyle \frac{1}{2}\left(3 + \frac{3}{4}y\right)^2 - \frac{1}{2}\left(\frac{3}{2} + \frac{3}{4}y\right)^2 - 4 = \frac{3}{2}y\)★★★☆☆
b)\(\displaystyle \frac{5}{2}\left(\frac{4}{5}y - \frac{5}{2}\right)^2 - \frac{5}{2}\left(\frac{4}{5}y + \frac{3}{2}\right)^2 + \frac{5}{3} = \frac{3}{2}y\)★★★★☆
c)\(\displaystyle \frac{1}{2}\left(1 + \frac{6}{5}n\right)^2 - \frac{1}{2}\left(\frac{6}{5}n - 6\right)^2 - \frac{2}{5}n - \frac{5}{2} = 0\)★★★★☆

Alle Gleichungen haben genau eine Lösung. Quadratische Terme kürzen sich heraus. ★ = Schwierigkeitsgrad (1–5 Sterne).
