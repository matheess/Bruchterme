Aufgabe 1

a)\(\displaystyle \frac{7}{6}\left(\frac{1}{2}k + 3\right)\left(\frac{4}{3}k - 2\right) + \frac{1}{2}k - 1 = \frac{7}{6}\left(\frac{1}{2}k + 3\right)\left(-\frac{5}{7} + \frac{4}{3}k\right) + \frac{7}{4}k - 1\)\(k = -\dfrac{9}{4}\) interaktiv
b)\(\displaystyle \frac{5}{4}\left(\frac{1}{2}n - \frac{2}{5}\right)^2 + 5\left(-\frac{2}{5} + \frac{1}{2}n\right) - \frac{3}{2} = \frac{5}{9}\left(\frac{3}{4}n + \frac{6}{5}\right)\left(\frac{3}{4}n - \frac{6}{5}\right) + \frac{5}{2}n\)\(n = -5\) interaktiv
c)\(\displaystyle \frac{1}{2}\left(\frac{3}{2}\left(k + 5\right) + \frac{7}{6}\left(k - 4\right)\right) - \frac{4}{3}\left(\frac{3}{2} + \frac{5}{4}k\right) = 0\)\(k = -\dfrac{7}{4}\) interaktiv

Aufgabe 2

a)\(\displaystyle \frac{5}{3}n - 6 + 4n = \frac{7}{4}\left(\frac{1}{2}n - \frac{3}{7}\right)\left(\frac{5}{7}n - 8\right) + \frac{2}{3} - \frac{5}{4}\left(\frac{2}{3}n - 4\right)\left(\frac{3}{4}n - 5\right)\)\(n = -\dfrac{7}{3}\) interaktiv
b)\(\displaystyle \frac{3}{5}\left(\frac{5}{3} + \frac{8}{7}x\right)\left(-\frac{4}{3} + \frac{4}{7}x\right) + \frac{2}{7}x - \frac{2}{3}x = \frac{3}{5}\left(\frac{5}{3} + \frac{8}{7}x\right)\left(-4 + \frac{4}{7}x\right) + 6 + \frac{2}{7}\)\(x = \dfrac{5}{2}\) interaktiv
c)\(\displaystyle \frac{5}{2}\left(\frac{2}{3}m + \frac{6}{5}\right)^2 - \frac{8}{5}\left(\frac{5}{6}m - \frac{7}{4}\right)\left(\frac{5}{6}m + \frac{7}{4}\right) = \frac{6}{5}m - \frac{8}{3} - \frac{\frac{3}{2}m - \frac{4}{5}}{7}\)\(m = -\dfrac{11}{3}\) interaktiv

Aufgabe 3

a)\(\displaystyle \frac{5}{4}\left(\frac{3}{5} + \frac{3}{2}z\right)\left(\frac{3}{5} - \frac{3}{2}z\right) + \frac{2}{3}\left(-\frac{8}{7}z - 1\right) = 5\left(1 + \frac{3}{4}z\right)^2 + \frac{4}{7}z - 7\)\(z = \dfrac{1}{10}\) interaktiv
b)\(\displaystyle \frac{2}{5}\left(\frac{3}{4}x + \frac{7}{4}\right)^2 + \frac{1}{2}x - \frac{3}{5} = \frac{6}{5}x + 3 + \frac{9}{10}\left(\frac{1}{2}x + \frac{1}{3}\right)\left(\frac{1}{2}x - \frac{1}{3}\right)\)\(x = \dfrac{13}{2}\) interaktiv
c)\(\displaystyle \frac{4}{5}\left(\frac{2}{7}z + \frac{3}{2}\right)\left(\frac{2}{7}z - \frac{3}{2}\right) + \frac{\frac{2}{5}z + \frac{1}{3}}{7} = \frac{1}{5}\left(\frac{4}{7}z + 4\right)^2 + \frac{5}{7}z - 6\)\(z = \dfrac{2}{3}\) interaktiv
