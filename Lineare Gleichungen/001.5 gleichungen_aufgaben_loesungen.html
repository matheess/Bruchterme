Aufgabe 1

a)\(\displaystyle 0 = \frac{7}{4}\left(\frac{1}{3}n - \frac{3}{7}\right)\left(\frac{5}{3}n - 1\right) + \frac{9}{8}n - 3 - \frac{7}{4}\left(\frac{1}{3}n - \frac{3}{7}\right)\left(4 + \frac{5}{3}n\right) - \frac{7}{3}n - \frac{3}{8}\)\(n = \dfrac{1}{11}\) interaktiv
b)\(\displaystyle \frac{4}{3}\left(-\frac{1}{2}k + \frac{9}{8}\right)\left(-\frac{1}{2}k - \frac{9}{8}\right) = 3\left(\frac{1}{3}k - \frac{7}{6}\right)\left(\frac{1}{3}k + \frac{7}{6}\right) + \frac{7}{4}k + \frac{4}{3} - \frac{5}{6}\left(\frac{7}{8}k - \frac{3}{4}\right)\)\(k = \dfrac{3}{7}\) interaktiv
c)\(\displaystyle \frac{7}{8}\left(\frac{7}{8}k - 3\right)\left(\frac{9}{7}k + \frac{5}{7}\right) + \frac{8}{5}k - \frac{3}{4} + \frac{7}{6} = \frac{7}{8}\left(-3 + \frac{7}{8}k\right)\left(\frac{9}{7}k + \frac{7}{5}\right) + \frac{3}{8}k\)\(k = -\dfrac{19}{6}\) interaktiv

Aufgabe 2

a)\(\displaystyle \frac{49}{3}\left(\frac{1}{2}p + \frac{1}{4}\right)\left(\frac{1}{2}p - \frac{1}{4}\right) + \frac{4}{7}p - 1 = \frac{4}{3}\left(\frac{7}{4}p - \frac{3}{8}\right)\left(\frac{7}{4}p + \frac{3}{8}\right) + \frac{-5 + \frac{3}{7}p}{8}\)\(p = \dfrac{7}{3}\) interaktiv
b)\(\displaystyle -\frac{1}{2}\left(\frac{5}{8}k + \frac{1}{4}\right)\left(\frac{2}{7}k - \frac{3}{7}\right) + 6k + \frac{1}{2} - \frac{9}{2}k = -\frac{3}{5} - \frac{2}{7}\left(\frac{3}{2} + \frac{5}{8}k\right)\left(\frac{1}{2}k - \frac{6}{5}\right)\)\(k = -\dfrac{2}{5}\) interaktiv

Aufgabe 3
