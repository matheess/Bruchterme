Aufgabe 1

a)\(\displaystyle \frac{2}{5}\left(\frac{5}{4}\left(y - \frac{5}{4}\right) + \frac{2}{7}\left(\frac{5}{4} + y\right)\right) = \frac{2}{3}\left(\frac{8}{5}y + \frac{8}{7}\right)\)\(y = -\dfrac{11}{4}\) interaktiv
b)\(\displaystyle \frac{6}{5}\left(\frac{3}{4}t + 2\right)\left(\frac{5}{6}t - \frac{7}{6}\right) + \frac{5}{3} = \frac{9}{7}\left(\frac{1}{3}t - \frac{2}{3}\right)\left(-\frac{5}{4} + \frac{7}{4}t\right) + \frac{1}{2}t + \frac{5}{2} + t\)\(t = \dfrac{19}{6}\) interaktiv
c)\(\displaystyle \frac{1}{2}\left(\frac{7}{3}y + 5\right)^2 = \frac{1}{2}\left(\frac{4}{3} + \frac{7}{3}y\right)\left(\frac{4}{3} - \frac{7}{3}y\right) + \frac{2}{3}y - 1 - 8\left(5 + \frac{7}{3}y\right)\)\(y = -\dfrac{11}{6}\) interaktiv

Aufgabe 2

a)\(\displaystyle \frac{1}{2}\left(\frac{4}{3}y + 3\right)\left(-5 + \frac{5}{2}y\right) + \frac{8}{3}y + \frac{7}{2} = \frac{1}{2}\left(\frac{4}{3}y + 3\right)\left(\frac{5}{2}y + 4\right) + \frac{2}{3}y + 3\)\(y = -\dfrac{13}{4}\) interaktiv
b)\(\displaystyle \frac{1}{3}\left(\frac{8}{3}\left(z - \frac{7}{6}\right) + \frac{7}{3}\right) - \frac{5}{6}\left(\frac{1}{3}\left(z + \frac{4}{3}\right) - 1\right) = 0\)\(z = -\dfrac{1}{3}\) interaktiv
c)\(\displaystyle \frac{5}{2}\left(\frac{2}{5}k + \frac{6}{5}\right)\left(\frac{2}{5}k - \frac{6}{5}\right) - 4k + 1 = \frac{3}{5}\left(-\frac{1}{3} + \frac{4}{3}k\right)\left(\frac{1}{2}k - \frac{8}{3}\right) + \frac{8}{3}k + 7\)\(k = -\dfrac{16}{7}\) interaktiv

Aufgabe 3

c)\(\displaystyle \frac{\frac{4}{7}\left(\frac{7}{4}x + 3\right) + \frac{7}{4}}{3} - \frac{1}{2}\left(\frac{5}{3}\left(\frac{3}{2} + x\right) + \frac{2}{3}\right) = 0\)\(x = -\dfrac{6}{7}\) interaktiv
