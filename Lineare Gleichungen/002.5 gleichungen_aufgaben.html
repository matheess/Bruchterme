Gleichungen – Aufgaben (Komplexität 5)
Bestimme die Lösung der Gleichung. Mache die Probe.

Aufgabe 1

a)\(\displaystyle \frac{8}{3}\left(1 + \frac{3}{8}n\right)\left(\frac{7}{2}n - 1\right) + \frac{3}{2}\left(2 + 2n\right) - \frac{8}{7}\left(\frac{7}{4}n + \frac{7}{2}\right)\left(\frac{7}{4}n - \frac{7}{2}\right) = \frac{2}{3}n + 9\)★★★★☆
b)\(\displaystyle \frac{9}{5}\left(\frac{2}{7}n + \frac{3}{2}\right)\left(\frac{3}{2}n + \frac{8}{3}\right) + \frac{\frac{1}{3}n + 2}{4} - 9 = \frac{6}{5}\left(\frac{9}{7}n - \frac{3}{2}\right)\left(\frac{1}{2}n + 2\right) + \frac{7}{3}n\)★★★★☆
c)\(\displaystyle \frac{5}{3}\left(\frac{1}{4}x + \frac{3}{2}\right)\left(\frac{1}{2}x + 9\right) + \frac{7}{4}x + \frac{3}{4} = \frac{5}{3}\left(\frac{1}{4}x + \frac{3}{2}\right)\left(\frac{1}{2}x + \frac{2}{3}\right) + \frac{8}{5}x + 8\)★★★★☆

Aufgabe 2

a)\(\displaystyle \frac{2}{5}\left(\frac{1}{2} + \frac{9}{5}p\right)\left(\frac{1}{2} - \frac{9}{5}p\right) - \frac{18}{5}\left(\frac{3}{5}p + \frac{3}{2}\right)\left(\frac{3}{5}p - \frac{3}{2}\right) + \frac{9}{5}p + 1 = \frac{5}{2}p + 7\)★★★★★
b)\(\displaystyle \frac{4}{3}\left(-\frac{5}{3} + \frac{3}{2}t\right)\left(\frac{9}{7}t + 3\right) - \frac{3 + \frac{1}{3}t}{4} = \frac{27}{7}\left(\frac{1}{2}t + 1\right)\left(\frac{4}{3}t + \frac{7}{3}\right) - \frac{\frac{1}{2}t + \frac{5}{2}}{3}\)★★★★★
c)\(\displaystyle \frac{7}{2}\left(\frac{4}{7} + \frac{3}{4}n\right)\left(\frac{4}{7} - \frac{3}{4}n\right) + \frac{7}{6}\left(\frac{5}{7}n - \frac{1}{2}\right) - \frac{3}{2}n + \frac{7}{2} = \frac{7}{8}\left(\frac{3}{2}n - \frac{6}{7}\right)^2\)★★★★★

Aufgabe 3

a)\(\displaystyle 0 = \frac{2}{3}y - \frac{9}{7} - \frac{5}{2}\left(\frac{4}{3}y + \frac{8}{3}\right)\left(-\frac{5}{7} + \frac{5}{7}y\right) + 5\left(\frac{2}{7}y - \frac{5}{7}\right)\left(\frac{8}{3} + \frac{5}{3}y\right) + 2y - 2\)★★★★★
b)\(\displaystyle \frac{3}{5}\left(-\frac{7}{6} + \frac{1}{2}y\right)\left(8 + \frac{8}{5}y\right) + \frac{4 + \frac{9}{4}y}{3} - \frac{8}{5}\left(\frac{1}{2}y + 3\right)\left(\frac{3}{5}y - \frac{3}{4}\right) - \frac{8}{3}y + 3 = 0\)★★★★★
c)\(\displaystyle \frac{4}{5}\left(\frac{3}{5}z + 3\right)\left(\frac{3}{2}z + \frac{3}{4}\right) + \frac{\frac{6}{5}z - \frac{2}{7}}{5} - \frac{-\frac{9}{2} + \frac{3}{5}z}{3} = \frac{1}{2}\left(-7 + \frac{6}{5}z\right)^2\)★★★★★

Alle Gleichungen haben genau eine Lösung. Quadratische Terme kürzen sich heraus. ★ = Schwierigkeitsgrad (1–5 Sterne).
