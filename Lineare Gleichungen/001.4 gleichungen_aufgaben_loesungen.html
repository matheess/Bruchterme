Aufgabe 1

a)\(\displaystyle \frac{7}{12}\left(-5 + \frac{4}{3}m\right)^2 + \frac{5}{3}m - \frac{3}{2} = \frac{7}{3}\left(\frac{2}{3}m - \frac{3}{2}\right)^2 + 7\left(\frac{2}{3}m - \frac{3}{2}\right)\)\(m = 3\) interaktiv
b)\(\displaystyle \frac{3}{2}\left(\frac{4}{3}\left(n + 7\right) + \frac{1}{3}\right) - \frac{3}{4}\left(\frac{7}{2}\left(n + 6\right) + \frac{2}{3}\right) = 0\)\(n = -\dfrac{14}{5}\) interaktiv
c)\(\displaystyle 0 = \frac{1}{2}\left(\frac{1}{3}\left(\frac{5}{4} + n\right) - \frac{7}{3}\right) - \frac{\frac{1}{3}\left(\frac{5}{4}n - \frac{1}{2}\right) - \frac{8}{3}}{3}\)\(n = \dfrac{1}{2}\) interaktiv

Aufgabe 2

a)\(\displaystyle \frac{5}{8}\left(\frac{1}{2} + \frac{1}{2}k\right)\left(\frac{2}{5}k + \frac{2}{5}\right) + \frac{3}{4}k - 1 = \frac{1}{2}\left(\frac{1}{2}k - 4\right)^2 + \frac{\frac{2}{3}k - \frac{3}{2}}{4}\)\(k = 3\) interaktiv
b)\(\displaystyle \frac{4}{5}\left(\frac{2}{3}y + 3\right)^2 + \frac{6}{5}\left(3y + 2\right) - \frac{8}{3}y = \frac{16}{5}\left(\frac{2}{3} + \frac{1}{3}y\right)^2 - \frac{8}{3}\)\(y = -4\) interaktiv
c)\(\displaystyle \frac{4}{3}\left(\frac{4}{3}\left(-6 + t\right) + \frac{1}{3}\left(t + \frac{3}{5}\right)\right) - \frac{3}{5}\left(\frac{7}{3}t - 5\right) = 0\)\(t = 9\) interaktiv

Aufgabe 3

a)\(\displaystyle \frac{1}{3}\left(\frac{3}{2} + \frac{7}{3}m\right)^2 = \frac{49}{3}\left(\frac{1}{3}m - \frac{5}{6}\right)\left(\frac{1}{3}m + \frac{5}{6}\right) - 7m + 4 + \frac{8}{5}m - \frac{1}{2}\)\(m = -\dfrac{10}{9}\) interaktiv
b)\(\displaystyle 0 = \frac{5}{6}\left(-\frac{3}{2} + \frac{5}{2}n\right)\left(\frac{8}{7}n + 7\right) + \frac{1}{3}n + 4 - \frac{5}{6}\left(-\frac{3}{2} + \frac{5}{2}n\right)\left(-\frac{8}{5} + \frac{8}{7}n\right) - \frac{3}{2}n + \frac{7}{6}\)\(n = \dfrac{1}{3}\) interaktiv
c)\(\displaystyle \frac{5}{7}\left(\frac{2}{3}x - \frac{8}{5}\right)\left(\frac{2}{3}x + \frac{8}{5}\right) - \frac{5}{12}\left(\frac{4}{3}x - 6\right)\left(\frac{4}{7}x + \frac{5}{6}\right) + \frac{2}{5}x - \frac{2}{5} = \frac{2}{3}x + 3\)\(x = \dfrac{9}{2}\) interaktiv
