Aufgabe 1

a)\(\displaystyle \frac{7}{6}\left(\frac{5}{6}\left(m + \frac{2}{5}\right) + \frac{5}{3}\right) = \frac{\frac{8}{3}\left(3 + \frac{7}{6}m\right) + 6}{4}\)\(m = 6\) interaktiv
b)\(\displaystyle 0 = \frac{5}{7}\left(\frac{7}{2}\left(k - \frac{5}{7}\right) - 1\right) - \frac{2}{3}\left(\frac{8}{3}\left(k + \frac{1}{3}\right) + \frac{3}{2}\right)\)\(k = \dfrac{17}{3}\) interaktiv
c)\(\displaystyle \frac{2}{5}\left(\frac{1}{2}t + 4\right)^2 - 5 = \frac{5}{8}\left(\frac{2}{5}t + \frac{6}{5}\right)\left(\frac{2}{5}t - \frac{6}{5}\right) + \frac{6}{5}t + 1\left(4 + \frac{1}{2}t\right)\)\(t = -17\) interaktiv

Aufgabe 2

a)\(\displaystyle \frac{7}{5}\left(\frac{7}{5}t - \frac{3}{2}\right)\left(\frac{7}{5}t + \frac{3}{2}\right) - 3t - 7 = \frac{7}{5}\left(\frac{3}{2} + \frac{7}{5}t\right)\left(\frac{3}{2} - \frac{7}{5}t\right) - 5t - \frac{3}{2}\)\(t = \dfrac{11}{4}\) interaktiv
b)\(\displaystyle \frac{2}{5}\left(\frac{1}{2}z + 3\right)^2 + \frac{\frac{1}{2}z - 2}{5} - \frac{1}{10}\left(\frac{4}{3}z - 2\right)\left(\frac{3}{4}z - \frac{7}{2}\right) = \frac{5}{3}z + 7\)\(z = 18\) interaktiv
c)\(\displaystyle 0 = \frac{3}{2}\left(\frac{3}{2}n - 5\right) - \frac{5}{2}\left(\frac{5}{7}\left(n - \frac{3}{2}\right) + \frac{2}{5}\left(-6 + n\right)\right)\)\(n = \dfrac{11}{5}\) interaktiv

Aufgabe 3

a)\(\displaystyle \frac{5}{7}\left(\frac{8}{5}t - 1\right)^2 + 7 = \frac{24}{7}\left(\frac{2}{5} + \frac{4}{5}t\right)\left(\frac{2}{3}t + \frac{7}{4}\right) + \frac{7}{4}t + 3\left(\frac{8}{5}t - 1\right)\)\(t = \dfrac{4}{7}\) interaktiv
b)\(\displaystyle 0 = \frac{3}{4}\left(\frac{5}{6} + \frac{3}{5}m\right)^2 + \frac{4}{7}m - \frac{3}{7} - \frac{3}{4}\left(-\frac{3}{5}m + \frac{1}{2}\right)\left(-\frac{3}{5}m - \frac{1}{2}\right) - \frac{8}{7}\left(-\frac{3}{2}m + 6\right)\)\(m = \dfrac{13}{6}\) interaktiv
c)\(\displaystyle \frac{3}{5}\left(\frac{5}{7}x + 7\right)^2 - \frac{3}{5}\left(-\frac{5}{7}x + \frac{2}{3}\right)\left(-\frac{5}{7}x - \frac{2}{3}\right) - 8 = \frac{5}{6}x - \frac{1}{2} + \frac{1}{2}x\)\(x = -\dfrac{19}{4}\) interaktiv
