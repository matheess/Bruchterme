Aufgabe 1

a)\(\displaystyle \frac{8}{3}x + 4 = \frac{8}{3}\left(\frac{1}{2}x + \frac{3}{2}\right)\left(\frac{3}{4}x + \frac{2}{5}\right) + \frac{4}{3}x + 4 - \frac{8}{3}\left(\frac{1}{2}x + \frac{3}{2}\right)\left(-3 + \frac{3}{4}x\right)\)\(x = -\dfrac{17}{4}\) interaktiv
b)\(\displaystyle \frac{3}{2}\left(\frac{2}{3} + \frac{7}{6}p\right)\left(\frac{2}{3} - \frac{7}{6}p\right) + \frac{\frac{5}{8}p + \frac{1}{2}}{6} = \frac{7}{6}\left(\frac{3}{4}p + 1\right)\left(\frac{1}{2} + \frac{7}{3}p\right) + \frac{5}{2}p + 3\)\(p = -\dfrac{3}{4}\) interaktiv
c)\(\displaystyle \frac{1}{2}m - \frac{5}{2} = \frac{3}{5}\left(\frac{2}{3}m + \frac{5}{2}\right)\left(-5 + \frac{3}{2}m\right) - \frac{1}{2}\left(\frac{3}{2}m - \frac{3}{2}\right)\left(\frac{4}{5}m - \frac{3}{2}\right) - m - 1\)\(m = 15\) interaktiv

Aufgabe 2

a)\(\displaystyle \frac{1}{2}\left(\frac{4}{3}z + \frac{4}{5}\right)\left(6 + \frac{5}{7}z\right) + \frac{9}{8}z + 9 = \frac{1}{2}\left(\frac{4}{3}z + \frac{4}{5}\right)\left(\frac{5}{7}z + \frac{2}{7}\right) + \frac{4}{3}z - 6\)\(z = -\dfrac{24}{5}\) interaktiv
b)\(\displaystyle \frac{3}{10}\left(\frac{3}{4}k - \frac{4}{3}\right)\left(\frac{3}{4}k + \frac{4}{3}\right) + \frac{-8 + \frac{2}{7}k}{2} = \frac{2}{5}\left(\frac{9}{8}k + \frac{7}{3}\right)\left(\frac{3}{8}k - 7\right) + \frac{\frac{9}{2}k + \frac{5}{2}}{7}\)\(k = -\dfrac{5}{7}\) interaktiv
c)\(\displaystyle \frac{25}{3}\left(\frac{3}{2} + \frac{3}{5}n\right)\left(\frac{3}{2} - \frac{3}{5}n\right) + \frac{3}{7}n - \frac{2}{7} = \frac{4}{3}\left(-\frac{1}{4} + \frac{3}{2}n\right)^2 + \frac{3}{2}\left(-n - \frac{7}{3}\right)\)\(n = \dfrac{16}{3}\) interaktiv

Aufgabe 3

a)\(\displaystyle \frac{7}{8}\left(\frac{3}{5}t - \frac{1}{2}\right)\left(\frac{3}{5}t + \frac{1}{2}\right) + \frac{9}{7}\left(\frac{7}{4}t - 7\right) - \frac{7}{8}\left(\frac{7}{2} + \frac{3}{5}t\right)\left(\frac{7}{2} - \frac{3}{5}t\right) - \frac{3}{2}t + \frac{1}{4} = 0\)\(t = -\dfrac{7}{3}\) interaktiv
b)\(\displaystyle \frac{7}{4}\left(\frac{4}{3} + \frac{2}{5}x\right)\left(\frac{4}{3} - \frac{2}{5}x\right) + \frac{\frac{3}{2}x - \frac{3}{2}}{7} - \frac{7}{4}\left(-\frac{2}{5}x + \frac{5}{3}\right)\left(-\frac{2}{5}x - \frac{5}{3}\right) - \frac{4}{7}x + \frac{8}{7} = 0\)\(x = \dfrac{15}{2}\) interaktiv
c)\(\displaystyle \frac{7}{2}\left(\frac{4}{3} + \frac{8}{7}y\right)\left(\frac{3}{4}y - 5\right) - \frac{25}{12}\left(\frac{4}{5} + \frac{6}{5}y\right)\left(\frac{4}{5} - \frac{6}{5}y\right) + \frac{3}{4}y - 4 - \frac{3}{4}y = -5\)\(y = -\dfrac{14}{11}\) interaktiv
