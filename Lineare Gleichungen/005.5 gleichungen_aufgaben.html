Gleichungen – Aufgaben (Komplexität 5)
Bestimme die Lösung der Gleichung. Mache die Probe.

Aufgabe 1

a)\(\displaystyle \frac{4}{7}\left(-\frac{5}{8} + \frac{5}{6}n\right)\left(\frac{2}{3}n - 3\right) + \frac{9}{5}n = \frac{4}{7}\left(\frac{5}{6}n - \frac{5}{8}\right)\left(\frac{2}{3}n + \frac{3}{2}\right) + \frac{5}{3}n - \frac{3}{2} + \frac{8}{5}\)★★★★☆
b)\(\displaystyle \frac{3}{7}\left(-\frac{5}{7}m + \frac{3}{2}\right)\left(-\frac{5}{7}m - \frac{3}{2}\right) + \frac{5}{7}\left(-\frac{6}{5} - \frac{5}{4}m\right) - \frac{1}{4}m = \frac{3}{7}\left(\frac{5}{7}m - \frac{2}{3}\right)\left(\frac{5}{7}m + \frac{2}{3}\right) - \frac{5}{4}\)★★★★☆
c)\(\displaystyle \frac{1}{3}\left(\frac{9}{5}t - \frac{9}{4}\right)\left(\frac{9}{5}t + \frac{9}{4}\right) + \frac{\frac{4}{5}t + \frac{9}{4}}{4} + 6 = \frac{3}{4}\left(\frac{6}{5}t + \frac{3}{2}\right)\left(\frac{6}{5}t - \frac{3}{2}\right) + \frac{5}{4}t\)★★★★☆

Aufgabe 2

a)\(\displaystyle \frac{1}{2}\left(-3 + \frac{3}{4}m\right)\left(\frac{2}{3}m + \frac{9}{7}\right) + \frac{7}{5}m - \frac{5}{6} = \frac{1}{2}\left(\frac{3}{4}m - 3\right)\left(5 + \frac{2}{3}m\right) + \frac{3}{4}m + \frac{7}{2}\)★★★★☆
b)\(\displaystyle 0 = \frac{7}{6}p + \frac{9}{4} - \frac{4}{5}\left(\frac{5}{7}p - \frac{3}{2}\right)\left(\frac{5}{7}p + \frac{3}{2}\right) + \frac{5}{9}\left(-\frac{6}{7}p + \frac{3}{2}\right)\left(-\frac{6}{7}p - \frac{3}{2}\right) - \frac{3}{4}p - \frac{9}{5}\)★★★★★
c)\(\displaystyle \frac{3}{2}\left(-\frac{7}{3} + \frac{1}{4}k\right)^2 = \frac{3}{8}\left(\frac{1}{2}k - \frac{4}{3}\right)\left(\frac{1}{2}k + \frac{4}{3}\right) + \frac{4}{3}k + 4 - \frac{\frac{2}{3}k - \frac{4}{3}}{8}\)★★★★★

Aufgabe 3

a)\(\displaystyle \frac{12}{5}\left(\frac{1}{4} + \frac{4}{5}n\right)^2 + \frac{\frac{8}{5}n + 7}{2} = \frac{3}{5}\left(\frac{8}{5}n - \frac{1}{3}\right)\left(\frac{8}{5}n + \frac{1}{3}\right) + \frac{\frac{3}{5} + \frac{1}{2}n}{5}\)★★★★★
b)\(\displaystyle \frac{9}{8}\left(-\frac{7}{2}n + \frac{4}{3}\right)\left(-\frac{7}{2}n - \frac{4}{3}\right) - \frac{49}{8}\left(-\frac{3}{2}n + \frac{8}{7}\right)\left(-\frac{3}{2}n - \frac{8}{7}\right) - \frac{3}{4}n - \frac{7}{2} = \frac{6}{7}n + 7\)★★★★★
c)\(\displaystyle \frac{1}{2}\left(\frac{3}{4} + \frac{5}{4}n\right)\left(\frac{3}{4} - \frac{5}{4}n\right) + \frac{\frac{3}{5}n - 7}{2} - \frac{\frac{1}{2}n - 6}{5} = \frac{1}{2}\left(\frac{5}{4}n - \frac{5}{4}\right)\left(\frac{5}{4}n + \frac{5}{4}\right)\)★★★★★

Alle Gleichungen haben genau eine Lösung. Quadratische Terme kürzen sich heraus. ★ = Schwierigkeitsgrad (1–5 Sterne).
