Gleichungen – Aufgaben (Komplexität 5)

Aufgabe 1

a)\(\displaystyle \frac{1}{3}\left(\frac{7}{2} + \frac{7}{2}m\right)\left(-3 + \frac{9}{7}m\right) + \frac{9}{4}m + \frac{9}{2} - \frac{9}{5}m = \frac{1}{3}\left(\frac{7}{2}m + \frac{7}{2}\right)\left(\frac{9}{7}m - \frac{7}{5}\right) + \frac{3}{2}\)\(m = \dfrac{4}{5}\) interaktiv
b)\(\displaystyle \frac{3}{2}\left(\frac{3}{2} + \frac{4}{3}y\right)\left(\frac{3}{2} - \frac{4}{3}y\right) + \frac{\frac{7}{4}y - 9}{8} = 24\left(\frac{1}{3}y + \frac{7}{6}\right)\left(\frac{1}{3}y - \frac{7}{6}\right) + \frac{5}{3}y + 5\)\(y = 16\) interaktiv
c)\(\displaystyle \frac{2}{5}\left(\frac{1}{4}n + 3\right)\left(\frac{7}{5}n + \frac{9}{8}\right) + \frac{7}{6}n - 6 = \frac{2}{5}\left(3 + \frac{1}{4}n\right)\left(\frac{7}{5}n - \frac{3}{8}\right) + \frac{3}{4}n - \frac{4}{5}\)\(n = 6\) interaktiv

Aufgabe 2

a)\(\displaystyle \frac{3}{7}\left(\frac{1}{2} + \frac{3}{2}t\right)\left(\frac{2}{3} + \frac{3}{2}t\right) - \frac{12}{7}\left(\frac{3}{4}t + \frac{3}{2}\right)\left(\frac{3}{4}t - \frac{3}{2}\right) - 6t - \frac{3}{4} = \frac{7}{4}t + \frac{8}{3}\)\(t = \dfrac{1}{12}\) interaktiv
b)\(\displaystyle \frac{3}{2}\left(\frac{9}{4}k + \frac{7}{6}\right)\left(\frac{9}{4}k - \frac{7}{6}\right) - \frac{-\frac{5}{8} + \frac{5}{4}k}{6} = \frac{27}{8}\left(4 + \frac{3}{2}k\right)^2 - \frac{\frac{5}{3}k + 2}{8}\)\(k = -\dfrac{11}{8}\) interaktiv
c)\(\displaystyle \frac{1}{2}\left(1 + \frac{1}{2}y\right)^2 + \frac{5}{4}\left(\frac{3}{2} + \frac{3}{2}y\right) = \frac{1}{2}\left(\frac{1}{2}y + \frac{3}{2}\right)\left(\frac{1}{2}y - \frac{3}{2}\right) + \frac{1}{2}y + 6\)\(y = \dfrac{4}{3}\) interaktiv

Aufgabe 3

a)\(\displaystyle \frac{6}{7}\left(\frac{9}{5}z - \frac{3}{7}\right)\left(\frac{9}{5}z + \frac{3}{7}\right) - \frac{6}{7}\left(-\frac{9}{5}z + \frac{3}{7}\right)\left(-\frac{9}{5}z - \frac{3}{7}\right) - \frac{1}{3}z + 2 = \frac{4}{3}z - \frac{5}{6}\)\(z = \dfrac{17}{10}\) interaktiv
b)\(\displaystyle \frac{1}{2}\left(\frac{7}{3}m + \frac{1}{2}\right)\left(\frac{7}{3}m - \frac{1}{2}\right) + \frac{\frac{8}{3} + \frac{1}{3}m}{3} = \frac{98}{9}\left(-\frac{1}{2}m + \frac{1}{2}\right)\left(-\frac{1}{2}m - \frac{1}{2}\right) + \frac{-\frac{1}{4} + \frac{5}{3}m}{2}\)\(m = 5\) interaktiv
c)\(\displaystyle \frac{21}{5}\left(\frac{7}{6}n + \frac{2}{5}\right)^2 + \frac{9}{5}n - 2 = \frac{7}{5}\left(\frac{7}{4}n - \frac{1}{2}\right)\left(\frac{7}{3}n + \frac{7}{5}\right) + \frac{1}{2}\left(8 + \frac{3}{5}n\right)\)\(n = \dfrac{6}{5}\) interaktiv
