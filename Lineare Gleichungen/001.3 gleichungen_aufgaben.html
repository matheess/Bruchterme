Gleichungen – Aufgaben (Komplexität 3)
Bestimme die Lösung der Gleichung. Mache die Probe.

Aufgabe 1

a)\(\displaystyle \frac{2}{3}\left(\frac{7}{3}\left(-\frac{1}{2} + x\right) - \frac{2}{3}\right) + \frac{1}{3} = 0\)★★★☆☆
b)\(\displaystyle \frac{5}{2}\left(\frac{1}{2}x - \frac{7}{5}\right)^2 - \frac{5}{2}\left(-\frac{6}{5} + \frac{1}{2}x\right)^2 = \frac{7}{2}x - \frac{7}{2}\)★★★☆☆
c)\(\displaystyle \frac{3}{2}\left(\frac{5}{6}m - \frac{2}{3}\right) + \frac{5}{2}\left(m + \frac{3}{2}\right) - \frac{1}{2} = 0\)★★★☆☆

Aufgabe 2

a)\(\displaystyle \frac{1}{2}\left(\frac{5}{4}t + 2\right)^2 + \frac{7}{5}t - 5 - \frac{7}{5}t = \frac{1}{8}\left(\frac{5}{2}t + 2\right)^2 - 5\)★★★☆☆
b)\(\displaystyle \frac{3}{4}\left(-\frac{1}{2} + \frac{3}{5}y\right)^2 + \frac{7}{6}y - 2 - \frac{2}{3}y = \frac{3}{4}\left(-\frac{5}{2} + \frac{3}{5}y\right)^2 - \frac{3}{4}\)★★★★☆
c)\(\displaystyle \frac{6}{5}\left(\frac{2}{3}y + \frac{3}{2}\right)^2 - 2y = \frac{24}{5}\left(-\frac{1}{2} + \frac{1}{3}y\right)^2 - 4y + \frac{5}{2} - 2\)★★★★☆

Aufgabe 3

a)\(\displaystyle \frac{3}{2}\left(\frac{1}{3}x + 4\right)^2 = \frac{3}{8}\left(6 + \frac{2}{3}x\right)^2 + \frac{7}{6}x + \frac{4}{3} + 2\left(\frac{1}{3}x + 4\right)\)★★★★☆
b)\(\displaystyle \frac{7}{3}n + 3 = \frac{5}{6}\left(\frac{7}{2} + \frac{5}{3}n\right)^2 - \frac{5}{6}\left(\frac{5}{3}n + \frac{1}{2}\right)^2\)★★★★☆
c)\(\displaystyle \frac{3}{2}\left(-2 + \frac{1}{3}m\right)^2 - m - \frac{2}{3} = \frac{2}{3}\left(\frac{1}{2}m + 2\right)^2 + \frac{2}{3}m + \frac{2}{3}\)★★★★☆

Alle Gleichungen haben genau eine Lösung. Quadratische Terme kürzen sich heraus. ★ = Schwierigkeitsgrad (1–5 Sterne).
