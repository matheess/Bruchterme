Gleichungen – Aufgaben (Komplexität 4)
Bestimme die Lösung der Gleichung. Mache die Probe.

Aufgabe 1

a)\(\displaystyle \frac{7}{6}\left(\frac{8}{7}t - 1\right)^2 - \frac{5}{2}t + \frac{7}{4} - \frac{96}{7}\left(\frac{1}{3}t - \frac{2}{3}\right)\left(\frac{1}{3}t + \frac{2}{3}\right) + \frac{5}{7}t - \frac{7}{3} = 0\)★★★★☆
b)\(\displaystyle \frac{4}{5}\left(\frac{3}{2}x + 5\right)\left(\frac{5}{6}x - \frac{2}{3}\right) - \frac{1}{2}\left(\frac{3}{2}x + \frac{2}{5}\right)\left(-1 + \frac{4}{3}x\right) - \frac{1}{2}x + \frac{5}{2} = \frac{3}{2}x - 2\)★★★★☆
c)\(\displaystyle \frac{\frac{1}{3}\left(8 + \frac{5}{7}x\right) + 5}{3} - \frac{4}{3}\left(\frac{3}{7}\left(-\frac{7}{6} + x\right) + \frac{4}{7}\right) = 0\)★★★★☆

Aufgabe 2

a)\(\displaystyle \frac{3}{2}\left(-\frac{2}{3} + \frac{2}{3}y\right)\left(-4 + \frac{3}{4}y\right) + \frac{7}{2}y + \frac{1}{2} = \frac{3}{2}\left(-\frac{2}{3} + \frac{2}{3}y\right)\left(\frac{3}{4}y - \frac{7}{6}\right) + \frac{4}{3}y - 8\)★★★★☆
b)\(\displaystyle 0 = \frac{5}{6}\left(\frac{4}{7}n + \frac{3}{4}\right) - \frac{1}{2}\left(\frac{1}{3}\left(n + \frac{7}{5}\right) + \frac{7}{6}\left(1 + n\right)\right)\)★★★★☆
c)\(\displaystyle \frac{75}{8}\left(\frac{7}{5} + \frac{1}{2}y\right)\left(\frac{7}{5} - \frac{1}{2}y\right) + \frac{5}{6}y + 2 = \frac{3}{2}\left(\frac{3}{2} + \frac{5}{4}y\right)^2 - \frac{3}{2}\left(\frac{5}{4}y + \frac{3}{2}\right)\)★★★★☆

Aufgabe 3

a)\(\displaystyle 0 = \frac{5}{4}\left(\frac{4}{3}\left(x + 6\right) - 3\right) - \frac{5}{2}\left(\frac{1}{3}\left(x + \frac{3}{2}\right) - \frac{2}{3}\right)\)★★★★☆
b)\(\displaystyle \frac{2}{5}\left(\frac{7}{2}t + \frac{1}{2}\right)^2 + \frac{5}{3}\left(-t - 2\right) - \frac{9}{10}\left(-\frac{7}{3}t + \frac{4}{3}\right)\left(-\frac{7}{3}t - \frac{4}{3}\right) - \frac{8}{3}t + 2 = 0\)★★★★★
c)\(\displaystyle \frac{7}{4}\left(\frac{5}{7}t + \frac{5}{7}\right)^2 = \frac{9}{7}\left(-\frac{5}{6}t + \frac{7}{3}\right)\left(-\frac{5}{6}t - \frac{7}{3}\right) + \frac{2}{3}t - 1 - \frac{\frac{6}{5}t + 7}{4}\)★★★★★

Alle Gleichungen haben genau eine Lösung. Quadratische Terme kürzen sich heraus. ★ = Schwierigkeitsgrad (1–5 Sterne).
